Features :

- Tags/Cscope
- Tal '\' alignment (\t)
- Omnicompletion
- Synthax highlighting and functions highlighting
- Refactor (\r)
- Mark (\m)
- VCS connection
- Alignement plugin
- etc !


Install :
```sh
git clone https://bitbucket.org/sebastien_chemin/vim-config ~/.vim
ln -s ~/.vim/vimrc ~/.vimrc
```
