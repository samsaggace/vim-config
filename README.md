Features :

- Tags/Cscope
- Tal '\' alignment (\t)
- Omnicompletion
- Synthax highlighting and functions highlighting
- Refactor (\r)
- Mark (\m)
- VCS connection
- Alignement plugin
- etc !


Install :
```sh
git clone --recursive https://github.com/samsaggace/vim-config.git ~/.vim
ln -s ~/.vim/vimrc ~/.vimrc
```
